### 🔧 Try It Yourself

```markdown
# Images in LiaScript

--{{0}}--  
Here are three ways to insert images in LiaScript.

## ✅ Basic Markdown Image
![Nature scene](https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=400)

--{{1}}--  
Use this when you need a simple image fast.

## 🎨 Styled HTML Image
<img src="https://images.unsplash.com/photo-1441974231531-c6227db76b6e?w=300" alt="Forest" style="width: 250px; border-radius: 10px;">

--{{2}}--  
Use HTML when you want control over image style and layout.

## 🖼️ Image Gallery with Flexbox
<div style="display: flex; gap: 10px;">
  <img src="https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=100" style="width: 100px; height: 100px; object-fit: cover;">
  <img src="https://images.unsplash.com/photo-1441974231531-c6227db76b6e?w=100" style="width: 100px; height: 100px; object-fit: cover;">
</div>

--{{3}}--  
Perfect for showing a group of images side-by-side.
```

🔑 **Tips**:  

- `![alt](url)` = easy and quick  
- `<img>` = more control  
- Use Flexbox for side-by-side image layouts
